Discord Chat Bot using Python

While I was playing games with my friends on the Hypixel Minecraft Server, I wanted an easy way to check my in-game stats and see how I compared to others. I therefore decided to make a Discord Chat Bot using Python. This bot lets users quickly view their game stats by executing a simple command.

The program starts with the user executing a Discord command with their Minecraft username as a parameter. The program then sends a request to Mojang’s API to fetch the player’s UUID (a unique identifier for Minecraft accounts). With this UUID, the program can then send a request to the Hypixel API to get the player’s data. A JSON response is then returned from the API request and the bot sorts through and displays the relevant information in a clear embed message in Discord.

After some testing of the Bot, I noticed that it was a rather annoying to keep typing your username everytime you wanted to run a command. To solve this I made a database that links Discord User IDs to Minecraft Player IDs. Now, you can execute the command without a parameter, and it will still show your game stats. And if you want to check someone else's stats, you can still just type in their username. This makes everything faster and easier for the user. You can see how it works in the GIF below.

I’ve also implemented buttons below the bot’s response to switch between different modes and edit the message.

Car Rental Booking System using Python

At the beginning of the year, I started to develop a Car Rental Booking System using Python, and the Tkinter library to create a GUI. The program features an simple interface, complete with dropdown menus, text boxes, and buttons for easy data input. To enhance the user experience further, I integrated a calendar library, allowing users to easily select the booking start and end dates using an calendar widget.

The process begins with the user inputting their booking requirements like seat count, vehicle type and their personal information. The program then refines the vehicle list to match these requirements. Once done, the available and fitting vehicles can be easily booked by the user.

To implement the booking system I used arrays to keep track of vehicles and bookings. One array stored vehicle details, while another linked booking dates and registration plates for easy identification. This setup allowed the ability to easily determine whether a vehicle is available or not and therefore ensures efficient management and smooth functionality.

I noticed a potential issue that might arise, potentially the user could accidentally select an booking `end_date` earlier than or on the same day as the booking `start_date`. To address this, I've developed a function that prevents the selection of a date preceding the `start_date`, and instead, it automatically sets the `end_date` to be one day after the booking `start date`.

PC Build for a Family Member

During my summer break, I undertook the task of building a new computer for a family member that was frustrated with the speed of their old PC. Their budget was approximately £500, and my objective was to construct a reliable and future-proof system capable of handling a variety of tasks.

In terms of components, I went with the AM4 Platform, specifically the Ryzen A520 Chipset, and the Ryzen 5 5600G processor. This APU features 6 high-speed cores, simultaneous multi-threading, and excellent integrated graphics, making it a great fit for most tasks, including light gaming and potential 1080p video editing. For storage, I opted for the Samsung 970 Evo Plus 2TB NVME drive because of its easy installation and significantly faster speeds compared to standard SATA SSDs. This choice not only ensures faster performance but also future-proofs the computer. To complete the build, I housed everything in the Corsair 4000D case, known for its cool design and excellent airflow.

The assembly process was quick and straightforward, largely because I didn't need to install a dedicated graphics card or any drives that aren't directly connected to the motherboard. In total, it took less than 30 minutes to complete and is a huge upgrade from their previous computer.
